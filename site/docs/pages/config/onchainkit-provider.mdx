# `<OnchainKitProvider />`

Provides the OnchainKit React Context to the app.

## Usage

```tsx
import { base } from 'viem/chains';
import { OnchainKitProvider } from '@coinbase/onchainkit';

const App = () => {
  return (
    <OnchainKitProvider chain={base}>
      <MyComponent />
    </OnchainKitProvider>
  );
};
```
